## ğŸ§  The AI Model

The TensorFlow model is trained using an 8-feature malicious behavior dataset:

| Feature           | Description                             |
|-------------------|---------------------------------------|
| cpu_usage         | CPU load from the tab                  |
| network_calls     | Number of outgoing network requests   |
| iframe_count      | Suspicious hidden frames               |
| refresh_rate      | Page reload loop detection             |
| dom_changes       | DOM mutation events                    |
| memory_usage      | JS heap size                          |
| script_injections | Unauthorized `<script>` insertions     |
| eval_calls        | Number of `eval()` invocations        |

After training, the model is exported to TensorFlow.js using:

tensorflowjs_converter --input_format=tf_saved_model malicious_tab_model tfjs_model


These files are then placed inside the `tfjs_model/` folder of the extension.
